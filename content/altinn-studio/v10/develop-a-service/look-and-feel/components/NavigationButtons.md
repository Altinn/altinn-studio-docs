## Properties

| Property | Description | Allowed Values |
|----------|-------------|----------------|
| `type` | Required. | `"NavigationButtons"` |
| `textResourceBindings` | See [Text resource bindings](#text-resource-bindings-textresourcebindings). |  |
| `showBackButton` | Shows two buttons (back/next) instead of just 'next'. |  |
| `validateOnNext` | See [Validate on next](#validate-on-next-validateonnext). |  |
| `validateOnPrevious` | See [Validate on previous](#validate-on-previous-validateonprevious). |  |

## Text resource bindings (`textResourceBindings`)

| Property | Description | Allowed Values |
|----------|-------------|----------------|
| `back` | Text on the back/previous page button |  |
| `next` | Text on the next page button |  |
| `backToPage` | Text on the "Back to Page" button when linkToPage/linkToComponent expression is used. |  |

## Validate on next (`validateOnNext`)

| Property | Description | Allowed Values |
|----------|-------------|----------------|
| `page` | Which pages should be validated when the next button is clicked. Required. | `"current"`, `"currentAndPrevious"`, `"all"` |
| `show` | List of validation types to show Required. |  |

## Validate on previous (`validateOnPrevious`)

| Property | Description | Allowed Values |
|----------|-------------|----------------|
| `page` | Which pages should be validated when the back button is clicked. Required. | `"current"`, `"currentAndPrevious"`, `"all"` |
| `show` | List of validation types to show Required. |  |
