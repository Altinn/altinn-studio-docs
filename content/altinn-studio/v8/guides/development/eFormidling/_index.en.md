There are two ways to configure eFormidling integration in an Altinn 3 application:

## eFormidling as a service task (recommended)

For applications using **version 8.9 or later**. Configuration is placed directly in the BPMN process definition.

[Read more about service task setup →](service-task/)

## eFormidling legacy setup

The original method with configuration in `applicationmetadata.json` and `appsettings.json`. Works for v8.0+, but will likely be removed in v9.0.

[Read more about legacy setup →](legacy/)
