---
title: Library Code Lists
description: What are library code lists?
weight: 10
---

Library code lists are code lists which are published using the Altinn Studio Designer organisation library.

To publish a code list

1. Navigate to [Altinn Studio Designer](https://altinn.studio/)
2. Choose the "library" tab
3. Follow the instructions to choose your organisation
4. Choose the codelist option from the resources menu
5. Create your code list
6. Save your code list
7. Publish your code list

You can now use the code list in your application by following the [shared codelist development guide](/en/altinn-studio/v8/guides/development/options/sources/shared-resources/)
