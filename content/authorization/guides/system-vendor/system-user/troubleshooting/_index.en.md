## Introduction

This guide is intended for vendors and developers to help diagnose and resolve common errors encountered when using the System User integration with Altinn. Most issues stem from configuration problems or a mismatch between the data sent in an API request and the information registered in Altinn.

Please find your error code in the list below and follow the link to a detailed troubleshooting guide.

## Common Error Codes

*   [**`invalid_altinn_customer_configuration (MP-303)`**](errors/mp-303-error/_index.en.md)
    *   This error indicates a configuration issue within Altinn's System Register for the specified system user.
