Authorization

Rights In Altinn Apps/Platform a right is a permission to perform an action on a given resource. The action can be one of some predefined option like read, write and sign but can also be any other action defined on a custom api in the apps. The resource is typical an app instance or part of it where data belongs to a given party (person or organisation). A user or system gets a right based on rules that describes permissions based on being a specific user or having a specific role. »
