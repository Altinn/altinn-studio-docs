Stylecop Analyzer

StyleCop.Analyzers provides warnings that indicate style and consistency rule violations in C# code. The warnings are organized into rule areas such as documentation, layout, naming, ordering, readability, spacing, and so forth. Each warning signifies a violation of a style or consistency rule. In altinn studio application, we have turned set this “warning” action to “Error” to enforce the use of rules.
